# 👥 고객 Segmentation

고객 데이터를 분석하고 세분화하는 모듈입니다.

## 📋 주요 기능

### 1. 고객 데이터 분석
- RFM 분석
- 행동 패턴 분석
- 거래 이력 분석
- 고객 가치 평가

### 2. 세그멘테이션
- 클러스터링
- 고객 프로파일링
- 세그먼트 정의
- 세그먼트 특성 분석

### 3. 마케팅 전략
- 타겟팅 전략
- 프로모션 설계
- 고객 이탈 예측
- 고객 생애가치 예측

## 🛠️ 사용 기술
- Python
- Pandas
- Scikit-learn
- K-means
- DBSCAN
- Matplotlib

## 📊 분석 구조
- 고객 분석
  - RFM 분석
  - 행동 분석
  - 가치 분석
- 세그멘테이션
  - 클러스터링
  - 프로파일링
  - 특성 분석
